# 🌌 Nova Portfolio  
HTML · CSS · JavaScript 기반 개인 포트폴리오

![preview](./assets/img/preview.png)  
> 실제 서비스 경험을 바탕으로, 실전형 풀스택 개발자를 지향하는 Nova의 포트폴리오 사이트입니다.  

---

## 🚀 프로젝트 소개

이 레포지토리는 **기본 스택(HTML/CSS/JavaScript)** 만으로 제작한  
개발자 **1인 포트폴리오 페이지**입니다.  

화려한 프레임워크 없이, 가장 기본 요소만으로도  
**세련된 UI·UX**를 만들 수 있다는 것을 보여주는 데 집중했습니다.

또한 Cursor 및 AI 기반 코딩 워크플로우를 학습하기 위한  
**리팩토링 실습 프로젝트**로도 활용합니다.

---

## 🧩 주요 특징

### ✔️ 1. 단일 페이지(SPA 느낌)의 깔끔한 레이아웃  
- Hero  
- About  
- Skills  
- Projects  
- Contact  
섹션으로 구성된 직관적인 구조입니다.

### ✔️ 2. 반응형 UI  
- 모바일/태블릿/데스크탑 지원  
- CSS만으로 구현된 반응형 레이아웃

### ✔️ 3. 간결하고 세련된 디자인  
- 다크 모드 기반의 네이비 그라데이션  
- 민트 포인트 컬러  
- 넉넉한 여백  
- 부드러운 호버/페이드 애니메이션 적용

### ✔️ 4. 기본 JavaScript만 사용  
- 부드러운 스무스 스크롤  
- 네비게이션 링크 하이라이트  
- 섹션 페이드인 효과  
- 필요 최소한의 인터랙션 구현

### ✔️ 5. Cursor 실습 프로젝트  
- 컴포넌트 분리  
- CSS 리팩토링  
- 반응형 개선  
- 코드 구조화 연습  

AI와 함께 개발하는 프로세스를 실험하는 데 적합합니다.
